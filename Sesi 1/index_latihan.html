Basic Learn Web for Beginers

Apa Itu Web?

Web adalah singkatan dari World Wide Web, yaitu bagian dari internet yang terdiri dari halaman-halaman yang dapat diakses melalui browser web. Web juga dikenal sebagai WWW atau W3.

Web Memiliki beberapa fungsi, di antaranya

  • Menghubungkan orang dengan informasi
  • Memungkinkan pengguna untuk mencari dan mengakses informasi dari berbagai sumber di seluruh dunia
  • Memungkinkan individu atau entitas untuk berkomunikasi dengan audiens online mereka

Website adalah kumpulan halaman web yang dapat diakses melalui internet. Website dapat berisi teks, gambar, video, dan berbagai jenis konten lainnya. Website dapat dibuat dan dikelola oleh individu, organisasi, ataupun bisnis.

Alamat web atau uniform resource locator (URL) adalah lokasi internet atau intranet dari sebuah situs web, file, atau basis data.

tiga bidang bahasa markup dan bahasa pemrograman yang di rekomendasikan kepada pemula untuk mempelajari membangun web, adalah sebagai berikut :

  1. HTML
  2. CSS
  3. Javascript

HTML (HyperText Markup Language)

adalah bahasa pengkodean yang digunakan untuk membuat halaman web dan aplikasi web. HTML merupakan standar internet yang digunakan secara luas untuk menampilkan halaman web.

CSS (Cascading Style Sheets)

yaitu bahasa pemrograman yang digunakan untuk menentukan tampilan dan format website. CSS berfungsi untuk mengatur gaya visual seperti warna, font, tata letak, dan dekorasi pada sebuah website.

Javascript

bahasa pemrograman yang digunakan untuk membuat halaman web yang interaktif dan dinamis. JavaScript merupakan salah satu teknologi inti dari World Wide Web dan bahasa pemrograman yang populer dalam pengembangan web.

Kesimpulan

HTML, CSS, dan JavaScript adalah tiga teknologi inti dalam pengembangan web yang saling melengkapi. HTML bertindak sebagai kerangka dasar sebuah halaman web, menentukan struktur dan kontennya. CSS kemudian mengambil alih peran dalam mengatur tampilan visual, seperti warna, tata letak, dan tipografi. JavaScript memberikan sentuhan interaktivitas, memungkinkan elemen-elemen dalam halaman web merespons tindakan pengguna. Bayangkan HTML sebagai kerangka rumah, CSS sebagai cat dan perabotnya, dan JavaScript sebagai sistem listrik dan pipa yang menghidupkan rumah tersebut. Ketiganya bekerja sama untuk menciptakan pengalaman pengguna yang dinamis dan menarik di dunia digital.
